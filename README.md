# Personal Portfolio

A modern developer portfolio built with **React** and **TailwindCSS** to showcase projects, skills, and professional experience.  
The project focuses on clean UI, responsive design, and efficient state management.

---

## Features

- ⚡ Fast and responsive UI built with **React**
- 🎨 Modern styling with **TailwindCSS**
- 🧠 Global state management using **React Context API**
- 💾 Persistent user preferences with **localStorage**
- 🔗 External data fetching with **Axios**
- 📱 Fully responsive design
- 🧩 Modular and reusable component structure

---

## 🛠️ Tech Stack

**Frontend**
- React
- TailwindCSS
- JavaScript (ES6+)

**State Management**
- React Context API

**Data Handling**
- Axios (API calls)
- localStorage (persistent client-side data)
